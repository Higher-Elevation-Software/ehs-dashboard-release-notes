# Introducing: Comprehensive ChemTrack Cache Performance Monitoring for Admins

**Date:** Dec 10, 2025  
**Category:** Feature

We're excited to announce the full deployment of our new **ChemTrack Cache Performance Monitoring** feature, providing administrators with critical insights directly within System Settings. This comprehensive solution includes a dedicated new page, integrating intuitive summary cards, a detailed cache metrics table, and insightful rollup statistics. Leveraging a new, robust backend API with real-time data fetching capabilities, administrators can now easily monitor vital ChemTrack cache lifecycle and performance metrics, ensuring optimal system operation and proactive management. This enhancement is fully integrated into the System Settings UI, offering a seamless experience for managing your EHS platform.
