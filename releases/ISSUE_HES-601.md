# Introducing: Comprehensive ChemTrack Cache Performance Monitoring for Admins

**Date:** Dec 10, 2025  
**Category:** Feature

We're excited to announce the full deployment of our new **ChemTrack Cache Performance Monitoring** feature, providing administrators with critical insights directly within System Settings. This comprehensive solution includes a dedicated new page, integrating intuitive summary cards, a detailed cache metrics table, and insightful rollup statistics. Leveraging a new, robust backend API with real-time data fetching capabilities, powered by a new React Query hook that includes periodic auto-refresh and company-level data scoping, administrators can now easily monitor vital ChemTrack cache lifecycle and performance metrics. This ensures optimal system operation and proactive management. The feature is seamlessly integrated into the System Settings UI with a new dedicated card and navigation, offering an effortless experience for managing your EHS platform.
